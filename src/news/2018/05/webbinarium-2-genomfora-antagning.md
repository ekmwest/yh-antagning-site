# Webbinarium: - Genomföra antaning

<!-- news-article-meta.html, { authorName: "Martin Tholsby", authorImage: "martin-tholsby.jpg", date: "15 maj 2018" } -->

Här går vi igenom hur ni genomför antagning i YH-antagning. Detta genomfördes som ett live-sänt webbinarium våren 2017. 

Innehåll:

- Inför antagningsbeslut
- Registrera antagningsbeslut
- Utskick av antagningsbesked / sista svarsdag
- Reservantagning
- Ändringar
- Upprop vid utbildningsstart
- Efterarbete
