# Driftstörnignar

<!-- news-article-meta.html, { authorName: "Martin Tholsby", authorImage: "martin-tholsby.jpg", date: "21 januari 2020" } -->

Igår (20:e januari) mellan kl 15:00 och 16:30 upplevde vissa sökande och administratörer problem när de besökte yh-antagning.se. Driftstörningarna berodde på ett flertal brott på fiberkablar vilket orsakade anslutningsproblem till yh-antagning.se i Norden. Kl 16:30 var yh-antagning.se tillgängligt som vanligt igen.
